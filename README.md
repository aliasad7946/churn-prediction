\# Customer Churn Prediction



\## Objective

Predict customer churn using machine learning to help businesses reduce customer loss.



\## Tools

Python (pandas, sklearn, matplotlib)



\## Model

Random Forest Classifier



\## Results

\- Accuracy: \~80–85% (varies by dataset)

\- Key predictive features include contract type, monthly charges, and tenure.



\## Key Insight

Customers with short-term contracts and higher monthly charges are more likely to churn.



\## Business Value

This model helps companies identify high-risk customers and take proactive retention actions.



\## Real-World Relevance

This project demonstrates practical application of predictive analytics in business decision-making and customer retention strategies.
